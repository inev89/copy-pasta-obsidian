```
aws iam create-user --username <username>

## Create Policy
```
{
    "Version": "2012-10-17",
    "Statement": {
        "Effect": "Allow",
        "Action": "sts:AssumeRole",
        "Resource": "arn:aws:iam::<your aws account id>:role/<your role name>"
    }
}
```

```
aws iam create-policy --policy-name LeakyBucket --policy-document file://labs-profile
```

## Attach Policy to User
```
aws iam attach-user-policy --user-name offsec --policy-arn arn:aws:iam::977522846777:policy/LeakyBucket
```

## Create Role

```
aws iam create-role --role-name MyCustomRole --assume-role-policy-document file://trust-policy.json
